Account and Data Deletion Request – CineBox - Dicas Filmes/Séries



To request the deletion of your account and all associated data from the CineBox - Dicas Filmes/Séries app, please send an email to abacattitech@gmail.com with the subject line:

"Data Deletion Request for CineBox - Dicas Filmes/Séries App"

Once your request is received, all user data will be permanently deleted from the application. This includes any personal information, usage history, and preferences stored within the app.

This process is managed by Abacatti Tech, the official developer of the app.
